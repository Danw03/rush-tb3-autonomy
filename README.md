# TB3 Autonomous Driving

TurtleBot3 Burger autonomous driving project using ROS 2 Jazzy.

## Packages

- tb3_lane_perception
- tb3_cone_perception
- tb3_reference
- tb3_mpc_controller
- tb3_safety
- tb3_autonomy_bringup

## Platform

- Raspberry Pi 5 8GB
- Ubuntu Server 24.04
- ROS 2 Jazzy
- TurtleBot3 Burger
